# 104.6 - ایجاد و تغییر لینک‌های سخت و نمادین

## اهداف یادگیری

در این فصل با موارد زیر آشنا می‌شوید:

- ایجاد لینک‌های سخت و نمادین
- تفاوت بین آنها

## کلیدواژه‌ها

`ln`, `hard link`, `symbolic link`

---

## لینک‌های سخت

لینک سخت اشاره‌ای مستقیم به inode فایل است.

```bash
$ ln target hardlink
```

هر دو فایل به یک داده اشاره می‌کنند. حذف یکی داده را حذف نمی‌کند.

## لینک‌های نمادین

لینک نمادین اشاره به نام فایل است.

```bash
$ ln -s target symlink
```

اگر target حذف شود، symlink شکسته می‌شود.

## تفاوت‌ها

- لینک سخت: همان inode، نمی‌تواند به دایرکتوری اشاره کند، نمی‌تواند cross-filesystem باشد.
- لینک نمادین: inode متفاوت، می‌تواند به هر چیز اشاره کند، می‌تواند cross-filesystem باشد.

## مدیریت

```bash
$ ls -l  # نوع لینک را نشان می‌دهد (l برای symlink)
$ readlink symlink  # هدف symlink را نشان می‌دهد
```
