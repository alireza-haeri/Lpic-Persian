# 104.7 - یافتن فایل‌های سیستم و قرار دادن فایل‌ها در مکان صحیح

## اهداف یادگیری

در این فصل با موارد زیر آشنا می‌شوید:

- درک ساختار FHS
- استفاده از ابزارهای جستجو
- قرار دادن فایل‌ها در مکان‌های صحیح

## کلیدواژه‌ها

`FHS`, `/usr`, `/var`, `/etc`, `find`, `locate`, `updatedb`

---

## FHS

Filesystem Hierarchy Standard ساختار دایرکتوری‌ها را تعریف می‌کند.

### دایرکتوری‌های مهم

- `/bin`: دستورات ضروری
- `/sbin`: دستورات سیستمی
- `/usr`: سلسله‌مراتب ثانویه
- `/var`: داده‌های متغیر
- `/etc`: تنظیمات
- `/home`: دایرکتوری کاربران
- `/root`: دایرکتوری root
- `/tmp`: فایل‌های موقت
- `/boot`: فایل‌های بوت

## جستجو

### find

```bash
$ find / -name filename
$ find /etc -type f -name "*.conf"
```

### locate

```bash
$ locate filename
$ updatedb  # به‌روزرسانی پایگاه داده
```

## قرار دادن فایل‌ها

- برنامه‌ها: `/usr/bin` یا `/usr/local/bin`
- تنظیمات: `/etc`
- لاگ‌ها: `/var/log`
- داده‌ها: `/var` یا `/usr/share`
