## استانداردهای ترجمه

تمام فایل‌های ترجمه شده از استانداردهای زیر پیروی می‌کنند:

### 1. زبان فارسی واضح و دوستانه
- متن به فارسی ساده و قابل فهم ترجمه شده
- از لحن معلم استفاده شده (نه رسمی و پیچیده)
- مناسب برای یادگیری و مطالعه

### 2. حفظ اصطلاحات فنی
- تمام دستورات Linux بدون تغییر باقی مانده‌اند
- نام فایل‌ها و مسیرها (paths) دست نخورده
- متغیرهای محیطی و پارامترها حفظ شده‌اند
- نام برنامه‌ها و ابزارها به انگلیسی باقی مانده‌اند

### 3. ساختار Markdown
- تمام سرفصل‌ها (headers) با ساختار # حفظ شده
- لیست‌های نقطه‌ای و شماره‌دار دست نخورده
- جداول با فرمت کامل
- بلوک‌های کد دقیقاً مانند اصل

### 4. مثال‌های عملی
- مثال‌های کاربردی اضافه شده
- توضیحات عملی برای دستورات مهم
- سناریوهای واقعی استفاده

### 5. خلاصه موضوعات
- خلاصه‌ای در پایان هر فایل
- مرور کلیدی‌ترین نکات
- یادآوری مفاهیم اصلی
