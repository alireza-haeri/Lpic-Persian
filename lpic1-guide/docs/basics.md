# مبانی لینوکس

در این بخش با مفاهیم پایه‌ای لینوکس آشنا می‌شوید.

## خط فرمان (Command Line)

خط فرمان یا Terminal محیطی است که به شما امکان می‌دهد با سیستم عامل از طریق دستورات متنی تعامل کنید.

### باز کردن Terminal

در اکثر توزیع‌های لینوکس می‌توانید با کلیدهای `Ctrl + Alt + T` ترمینال را باز کنید.

## دستورات اولیه

### pwd - نمایش مسیر جاری

دستور `pwd` (Print Working Directory) مسیر دایرکتوری فعلی را نمایش می‌دهد:

```bash
pwd
```

خروجی:
```
/home/username
```

### ls - لیست فایل‌ها و دایرکتوری‌ها

دستور `ls` محتویات یک دایرکتوری را نمایش می‌دهد:

```bash
# نمایش ساده
ls

# نمایش با جزئیات
ls -l

# نمایش فایل‌های مخفی
ls -a

# ترکیب پارامترها
ls -lah
```

### cd - تغییر دایرکتوری

با دستور `cd` (Change Directory) می‌توانید بین دایرکتوری‌ها جابجا شوید:

```bash
# رفتن به دایرکتوری خانگی
cd ~

# رفتن به دایرکتوری والد
cd ..

# رفتن به مسیر مشخص
cd /var/log

# بازگشت به دایرکتوری قبلی
cd -
```

## ساختار دایرکتوری‌های لینوکس

لینوکس از ساختار درختی فایل‌ها استفاده می‌کند که از ریشه `/` شروع می‌شود:

| دایرکتوری | توضیحات |
|-----------|----------|
| `/` | دایرکتوری ریشه (Root) |
| `/home` | دایرکتوری‌های خانگی کاربران |
| `/etc` | فایل‌های پیکربندی سیستم |
| `/var` | فایل‌های متغیر (لاگ‌ها، کش، و غیره) |
| `/bin` | فایل‌های اجرایی اصلی |
| `/usr` | برنامه‌ها و کتابخانه‌های کاربر |
| `/tmp` | فایل‌های موقت |

## کار با فایل‌ها

### cat - نمایش محتوای فایل

```bash
cat filename.txt
```

### touch - ایجاد فایل خالی

```bash
touch newfile.txt
```

### mkdir - ایجاد دایرکتوری

```bash
# ایجاد یک دایرکتوری
mkdir mydirectory

# ایجاد چند دایرکتوری تو در تو
mkdir -p parent/child/grandchild
```

### cp - کپی کردن

```bash
# کپی فایل
cp source.txt destination.txt

# کپی دایرکتوری
cp -r sourcedir/ destdir/
```

### mv - جابجایی یا تغییر نام

```bash
# تغییر نام
mv oldname.txt newname.txt

# جابجایی
mv file.txt /path/to/destination/
```

### rm - حذف کردن

```bash
# حذف فایل
rm file.txt

# حذف دایرکتوری
rm -r directory/

# حذف اجباری
rm -rf directory/
```

⚠️ **هشدار:** دستور `rm -rf` بسیار قدرتمند است و فایل‌ها را بدون تایید حذف می‌کند. با احتیاط استفاده کنید!
